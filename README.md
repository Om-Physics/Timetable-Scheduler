# Python Timetable Scheduler

A Python-based program to create and manage school or college schedules. 
Supports automatic timetable generation, conflict detection, and visualization.

## Features
- Create weekly schedules for classes, labs, and study sessions
- Avoid time conflicts between courses
- Export schedules to CSV, JSON, or PDF
- Optional GUI interface for easy usage

## Directory Structure
- `scripts/`: Core Python scripts for generating and managing timetables
- `data/`: Sample schedules and data files
- `notebooks/`: Notebooks for testing and visualization
- `gui/`: Graphical interface scripts (Tkinter, PyQt, Streamlit)
- `docs/`: Instructions and user guide
